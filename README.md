# WhatsApp Second Brain Bot 🧠

Este proyecto es un bot de asistencia personal integrado directamente en WhatsApp, desarrollado en Python. 

## Arquitectura
1. **Frontend:** WhatsApp API (para enviar y recibir mensajes).
2. **Backend:** Python con FastAPI/Flask.
3. **Cerebro AI:** Integración con LLMs (OpenAI/Anthropic) o LangChain.
4. **Almacenamiento y Tareas:** Google Calendar API.

## Pasos de Instalación Próximos
1. Obtener archivos `credentials.json` de Google Cloud Console.
2. Obtener Token de Twilio/Meta.
3. Instalar librerías (`pip install -r requirements.txt`).
